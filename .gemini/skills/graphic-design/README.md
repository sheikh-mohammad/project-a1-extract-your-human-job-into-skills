# Graphic Design Skill

This skill creates modern, aesthetic visual designs using Google Gemini image generation via MCP Playwright browser access.

## Overview

The skill generates high-quality design images, downloads them, and organizes them locally in the graphic-designs directory following modern aesthetic principles. It's designed for creating professional graphics for tech platforms, AI tools, learning products, and personal portfolios.

## Key Features

1. **Gemini Integration**: Uses MCP Playwright to access Google Gemini for image generation
2. **Modern Aesthetics**: Creates designs following professional design principles
3. **Image Download & Organization**: Downloads generated images and saves them in organized directory structure
4. **Prompt Construction**: Converts user requests into detailed, structured image prompts
5. **Professional Output**: Generates images suitable for tech platforms, AI tools, and professional use

## Aesthetic Guidelines

The generated designs follow these important principles:

### Must Have:
- Modern and contemporary approach
- Clean and uncluttered design
- Minimal and purposeful elements
- Professional appearance
- High visual hierarchy
- Balanced spacing
- Readable text areas
- Soft shadows for depth
- Subtle gradients
- Muted or neutral color palettes

### Avoid:
- Cartoon styles
- Loud or neon colors
- Overcrowded layouts
- Low-contrast text
- Generic clip-art visuals

## Usage

When prompted to create a design, the skill will:

1. Analyze the specified design purpose or requirements
2. Construct a detailed image prompt based on the requirements
3. Access Google Gemini via MCP Playwright browser
4. Generate 1-3 high-quality images
5. Download the images to the graphic-designs directory
6. Provide a summary of the generated images

## Output Format

The skill provides:
- Downloaded images saved in the graphic-designs directory
- Descriptive filenames based on the design purpose
- Summary of generated images with file names
- Location where files are saved
