# UI Design Skill

This skill generates modern, aesthetic, production-quality frontend UI using HTML, TailwindCSS, and minimal JavaScript.

## Overview

The skill creates clean, professional UI components and pages that follow modern design principles and best practices. It focuses on semantic HTML, utility-first styling with TailwindCSS, and minimal JavaScript for interactions.

## Key Features

1. **Semantic HTML**: Creates well-structured, accessible HTML markup
2. **TailwindCSS Styling**: Implements modern styling using utility-first approach
3. **Minimal JavaScript**: Includes only necessary JavaScript for interactions
4. **Modern Aesthetics**: Follows contemporary design principles and visual guidelines
5. **Production Quality**: Generates clean, readable, well-structured code

## Aesthetic Guidelines

The UI follows these important design principles:

### Must Feel:
- Modern and professional
- Minimal but not empty
- Well-spaced with generous padding
- Strong visual hierarchy
- Readable typography
- Balanced layout
- Soft shadows and rounded corners
- Subtle gradients or muted colors
- Consistent spacing system
- Clear section separation

### Avoid:
- Loud or neon colors
- Crowded layouts
- Overly complex animations
- Outdated UI patterns
- Bootstrap-style card clutter

## Usage

When prompted to create a UI, the skill will:

1. Analyze the specified page purpose or requirements
2. Apply modern design principles and aesthetic guidelines
3. Generate a complete HTML file with TailwindCSS styling
4. Include minimal JavaScript if needed for interactions
5. Output clean, production-ready code

## Output Format

The skill provides:
- Complete HTML file with embedded TailwindCSS CDN
- Minimal JavaScript inside `<script>` tag if needed
- Short comments explaining key sections
- Clean, readable, well-structured code
