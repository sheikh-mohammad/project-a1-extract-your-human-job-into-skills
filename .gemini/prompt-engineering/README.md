# Prompt Engineering Skill

This skill transforms raw prompts into clear, effective prompts using proven frameworks and best practices. It applies the six-part prompting framework and other advanced techniques to enhance prompt quality, specificity, and effectiveness.

## Overview

The skill enhances raw prompts by transforming them into clear, structured prompts using industry-standard frameworks and best practices. It leverages the six-part prompting framework and other advanced techniques to improve prompt effectiveness and output quality.

## Key Features

1. **Framework Application**: Applies the six-part prompting framework to structure prompts effectively
2. **Best Practice Integration**: Incorporates proven prompt engineering techniques
3. **Quality Enhancement**: Improves clarity, specificity, and effectiveness of prompts
4. **Template Generation**: Creates professional-grade prompts with optimal structure

## The Six-Part Framework

The skill implements the comprehensive six-part framework:

1. **Command**: Start with strong, clear directives
2. **Context**: Provide comprehensive background information
3. **Logic**: Define the output structure and reasoning approach
4. **Roleplay**: Assign specific expertise to the AI
5. **Formatting**: Structure information for success
6. **Questions**: Include iterative refinement questions

## Usage

When prompted to enhance a raw prompt, the skill will:

1. Analyze the input prompt for clarity and completeness
2. Apply the six-part framework to structure the prompt
3. Integrate best practices for the target domain
4. Generate an enhanced, professional-grade prompt
5. Provide explanations for the improvements made

## Best Practices Applied

- Use specific action verbs
- Provide comprehensive context
- Define output structure explicitly
- Assign expert roles to the AI
- Use appropriate formatting
- Include iterative refinement questions
